# Nets Easy - iOS SDK v1.6.2 (Swift Package Manager)


This is dedicated repository for Swift Package Manager (SPM) distribution for the Nets Easy - iOS SDK . It provides tagged, binary releases for easy and lightweigt integration.

If you are looking for the main repository, issue tracker and documentation, please visit [our main repository](https://github.com/Nets-eCom/Nets-Easy-iOS-SDK).

## Installation

Open your project in Xcode.

Go to File > Add Package Dependencies....

Enter the repository URL:

[https://github.com/Nets-eCom/Nexi-Checkout-iOS-SDK-spm](https://github.com/Nets-eCom/Nexi-Checkout-iOS-SDK-spm)


Select the version you wish to use (e.g., Up to Next Major Version from 1.6.2).
